Стандарты ведения проекта:

# docs
папка в которой лежат знания по проекту
# actors
папка в которой лежат экторы. Любая связка скрипт+сцена это точно эктор. Еще эктором может считаться что-то, что может в теории появиться на сцене. Концептуально это близко к префабам. Кажды эктор должен лежать в своей отдельной папке
# scripts
технические скрипты, которые не экторы, например утилиты, сервисы
# scenes
большие сцены. Main сцена, меню сцена, мир сцена и всякое такое
